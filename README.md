# IMC Project App

A flutter project for IMC calculation.

## Getting Started

You need have flutter 3.10.2 or higher installed in your machine.

## How to run

```bash
flutter run
```

## How to build

```bash
flutter build apk
```

## How to test

```bash
flutter test
```

## How to lint

```bash
flutter analyze
```

## How to format

```bash
flutter format .
```
